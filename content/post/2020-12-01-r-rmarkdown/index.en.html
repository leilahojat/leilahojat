This site was created using the Wowchemy website builder and Hugo static site generator, with deployment by Netlify and hosting on GitHub. Coding was done in Rstudio using the blogdown package. More information and instructions on using Wowchemy can be found on their website. Example R studio code for the Wowchemy starter-academic template can be found on GitHub. My subdomain name was obtained through Rbind support by filing an issue on GitHub. Additionally, the original site development closely followed the YouTube CradleToGraveR tutorial, which can be found at this link.

Thanks for visiting my website, and look out for ongoing updates!
